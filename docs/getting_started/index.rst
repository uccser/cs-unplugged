Getting Started
##############################################################################

This documentation will help you understand how the project is setup,
basic steps on how to use it, and our guidelines for your contributions.

.. note::

  This project adheres to the Contributor Covenant code of conduct.
  By participating, you are expected to uphold this code.
  Please read our :doc:`Code of Conduct <code_of_conduct>` before continuing.
  You can report unacceptable behaviour by
  `emailing us <mailto:csse-education-research@canterbury.ac.nz>`_.
